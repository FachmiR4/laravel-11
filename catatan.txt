cmd /> php artisan serve -> untuk menjalankan laravel
cmd /> php artisan make:controller <nama_folder>/<nama_file> -> untuk membuat controller
cmd /> php artisan make:migration create_<nama_file>_table -> untuk membuat file migration
cmd /> php migrate -> berfungsi untuk memasukan skema migrate ke database
cmd /> php artisan make:controller <nama_folder>/<nama_file> --resource -> untuk membuat controller yang sudah lengkap dengan kebutuhan
cmd /> php artisan make:model <nama_file> -> untuk membuat file model baru
